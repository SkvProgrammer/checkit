break and continue

Share Tweet Youtube Language

break and continue are flow control statements for looping/iteration statements...Loops iterate over a block of code until test expression is false, but sometimes we wish to terminate the current iteration or even the whole loop without checking test expression.

break keyword in python terminates a loop without checking the test expression.It can be very handy in some cases...


for i in "Programming": if i == "m": break print(i)

Output of the Above Code

P
r
o
g
r
a

Continue

The continue statement is used to skip the rest of the code inside a loop for the current iteration only. Loop does not terminate but continues on with the next iteration.

We will first try to write a program without using the continue keyword and we will also write the same piece of code with a continue statement...

print("Without continue statement") #without continue statement for i in "Programming": if i == "m": print(i) print("Hello") print("With continue statement") #with continue statement for i in "Programming": if i == "m": print(i) continue print("Hello")

Output of the Above Code

Without continue statement
m
Hello
m
Hello
With continue statement
m
m

Installing Python

Share Tweet Youtube Language

Many Operating Systems have python already installed.So,if you want to check the installation follow the steps given below

If you are using Windows,start cmd(command prompt) from search menu or run command and type the following command

C:\Users\Your Name>python - -version

If you are using Linux or mac, type the following command in terminal

python - -version

If python is installed in your device the above command will show the version of python which is installed in your device,if it's showing some error it means python is not installed in your device you can download it from python's official website python.org

Now while installing python please tick the set path option in the installation wizard and if you forgot to tick it,you have to manually set it to path by copying the location where python is installed and then you have to go to This PC -> System Properties -> Advanced System Setting -> Environment Variables -> path -> paste the address of directory where python is installed(this is just for windows)

Note:Remember one thing that python has basically two major versions that are Python 2.x and Python 3.x both have few difference in the syntax,Python 3.x is very widely used and I will recommend you to install Python 3.x version as we are going to cover Python 3.x in this course

If you want to setup environment variables in Linux type the following command in terminal

export PATH=$PATH:/Path of Python

If you want to setup environment variables in Mac type the following command in terminal

~/.bash_profile

The above command will open the file .bash_profile,if no such file is there,then this nano command will create the file .bash_profile

export PYTHON_HOME = Location of Python

After installation and path set up is completed you have two windows to work with,the first one is python's command line and other is Interactive Shell,First We will look at the command line of python by typing the following cmd or terminal

python

On typing this you will get the following appearance in the cmd or terminal

C:\Users\Your Name>python
Python 3.6.8 (tags/v3.6.8:3c6b436a57, Dec 24 2018, 00:16:47) [MSC v.1916 64 bit (AMD64)] on win32
Type "help", "copyright", "credits" or "license" for more information.
>>>

Now we will try to make our first python program in command line i.e Hello World Program

C:\Users\Your Name>python
Python 3.6.8 (tags/v3.6.8:3c6b436a57, Dec 24 2018, 00:16:47) [MSC v.1916 64 bit (AMD64)] on win32
Type "help", "copyright", "credits" or "license" for more information.
>>>print("Hello World")
Hello World
>>>

As you can see we have made our first python program but wait is it a good experience while writing a code , Absolutely no We need a particular software with graphical interface for programming i.e called an IDE(Interactive Development Environment),We will discuss about it in the next chapter.
Python provides us a build-in editor for Graphical interface to have a good experience i.e called IDLE(Interative Development and Learning Environment),which is not very powerful but a far better than the command line approach
